# Alex Teng Studio

Premium one-page personal portfolio website for Alex Teng, focused on planter and gardenware product design studies.

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion

## Local Setup

Install dependencies:

```bash
pnpm install
```

Run the development server:

```bash
pnpm dev
```

Open `http://localhost:3000`.

## Build

```bash
pnpm build
```

Preview the built app:

```bash
pnpm start
```

## Deploying to Vercel

1. Import this repository into Vercel.
2. Keep the framework preset as Next.js.
3. Use the default build command: `pnpm build`.
4. Use the default output settings.
5. Deploy from the `main` branch.
